## 機能一覧
 ・打刻機能  <br>
  「勤怠開始」「勤怠終了」「休憩開始」「休憩終了」の情報を記録する  <br>
 ・勤怠情報確認機能  <br>
   日付別に勤怠情報を確認する  <br>
 ・ログイン機能（メール認証有）  <br>
